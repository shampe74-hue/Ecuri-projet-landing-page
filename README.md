# Ecuri Landing Page

Landing page responsive développée en HTML, CSS, JavaScript et PHP.

## Fonctionnalités

- Hero section moderne
- Design responsive
- Formulaire de contact
- Envoi d’e-mails avec PHPMailer
- Animations JavaScript

## Technologies utilisées

- HTML5
- CSS3
- JavaScript
- PHP
- PHPMailer
- Ajax

## Installation

1. Cloner le repository
2. Ouvrir le projet dans VS Code
3. Configurer WAMP
4. Configurer les informations SMTP dans le fichier PHP
5. Lancer le projet en localhost
